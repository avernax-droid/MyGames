Documento de Implementação: Sistema MyGames (V2)
Este documento consolida a arquitetura, regras de negócio e scripts necessários para a implementação da segunda versão do sistema de recompra de produtos usados. Mantendo a abordagem Design First, toda a modelagem de dados e fluxo visual foi estruturada antes da codificação.


1. Estrutura e Separação de Ambientes
Para o desenvolvimento no Windows 11 utilizando o VS Code, o projeto usará variáveis de ambiente e bancos de dados separados. Isso garante que as alterações não impactem a versão demo apresentada ao cliente.

2. Modelagem do Banco de Dados (MySQL)
As tabelas foram reestruturadas para tornar as categorias dinâmicas e suportar os fluxos de perguntas de avaliação e tratativas manuais ("Outros" e "Jogos").
2.1 Criação das Novas Tabelas Base
CREATE TABLE categorias (
   id INT AUTO_INCREMENT PRIMARY KEY,
   nome VARCHAR(50) NOT NULL,
   ativo TINYINT(1) DEFAULT 1
);
INSERT INTO categorias (nome) VALUES ('Console'), ('Controle'), ('Acessório'), ('Jogo');

CREATE TABLE perguntas_conservacao (
   id INT AUTO_INCREMENT PRIMARY KEY,
   categoria_id INT NOT NULL,
   texto_pergunta VARCHAR(255) NOT NULL,
   tipo_resposta ENUM('Sim/Nao', 'Multipla_Escolha') NOT NULL,
   impacto_valor DECIMAL(10,2) DEFAULT 0.00,
   FOREIGN KEY (categoria_id) REFERENCES categorias(id)
);

CREATE TABLE avaliacoes_manuais (
   id INT AUTO_INCREMENT PRIMARY KEY,
   nome_produto_informado VARCHAR(255) NOT NULL,
   descricao_estado TEXT,
   email_contato VARCHAR(150) NOT NULL,
   data_solicitacao DATETIME DEFAULT CURRENT_TIMESTAMP,
   status ENUM('Pendente', 'Avaliado', 'Recusado') DEFAULT 'Pendente'
);

CREATE TABLE propostas_venda (
   id INT AUTO_INCREMENT PRIMARY KEY,
   usuario_id INT,
   produto_id INT NULL,
   categoria_id INT NOT NULL,
   quantidade INT DEFAULT 1,
   observacoes TEXT,
   fotos_caminho VARCHAR(500),
   valor_preliminar DECIMAL(10,2),
   status_avaliacao ENUM('Pendente', 'Em Análise Física', 'Concluída') DEFAULT 'Pendente',
   data_proposta DATETIME DEFAULT CURRENT_TIMESTAMP,
   FOREIGN KEY (categoria_id) REFERENCES categorias(id)
);

3. Lógica do Fluxo Visual e Interface (UI/UX)
O fluxo construído com os templates do Flask seguirá uma navegação em funil:
* Passo 1 (Seleção de Categoria): Exibição de blocos com as categorias ativas.
* Passo 2 (Seleção de Produto): Lista de produtos do catálogo (com foto e nome) e o card "Outros" no final.
* Passo 3 (Regras de Avaliação):
Cenário
	Comportamento do Sistema
	Consoles, Controles e Acessórios
	Carrega as perguntas da tabela perguntas_conservacao, calcula o desconto baseado no estado e exibe o preço final.
	Jogos
	Bypass nas perguntas. Solicita quantidade, observações e fotos. Exibe aviso de que a avaliação final será por e-mail.
	Outros
	Abre formulário limpo para envio de dados à tabela avaliacoes_manuais. Cotação será enviada por e-mail.
	

4. Script de Importação de Dados (Python)
O script abaixo utiliza a biblioteca pandas para processar a planilha de preços e popular a tabela catalogo_mestre do banco de desenvolvimento, substituindo o uso da API RAWG.
import pandas as pd
import mysql.connector
from mysql.connector import Error
import os
from dotenv import load_dotenv

load_dotenv('.env.dev')

db_config = {
   "host": os.getenv("DB_HOST", "127.0.0.1"),
   "port": 3307, 
   "user": os.getenv("DB_USER", "root"),
   "password": os.getenv("DB_PASS", "TonMix#25"),
   "database": os.getenv("DB_NAME", "mygames_dev"),
   "auth_plugin": "mysql_native_password"
}

def limpar_valor(valor):
   try:
       if isinstance(valor, str):
           valor = valor.replace('R$', '').replace(' ', '').replace(',', '.')
       return float(valor)
   except (ValueError, TypeError):
       return 0.0

def obter_mapa_categorias(cursor):
   cursor.execute("SELECT id, nome FROM categorias")
   return {nome.lower(): id_cat for id_cat, nome in cursor.fetchall()}

def popular_banco():
   conn = None
   try:
       df = pd.read_excel('Troca Game Avaliação - Geral.xlsx', sheet_name='2025', header=None)
       conn = mysql.connector.connect(**db_config)
       cursor = conn.cursor()
       print(f"Conectado ao MySQL na porta {db_config['port']}. Iniciando carga...")

       mapa_categorias = obter_mapa_categorias(cursor)
       blocos = [(0, 'Sony'), (5, 'Microsoft'), (10, 'Nintendo')]
       total_inserido = 0

       for col_inicio, plataforma in blocos:
           sub_df = df.iloc[4:, col_inicio:col_inicio+4]
           sub_df.columns = ['nome', 'credito', 'pix', 'venda']
           
           for _, row in sub_df.iterrows():
               nome = str(row['nome']).strip()
               
               if not nome or nome == 'nan' or any(term in nome for term in ["Valor", "Preço", "Analise"]):
                   continue
               
               nome_lower = nome.lower()
               termos_acessorio = ['cabo', 'headset', 'volante']
               termos_controle = ['controle', 'joy-con', 'dualshock']
               
               if any(word in nome_lower for word in termos_controle):
                   categoria_id = mapa_categorias.get('controle')
               elif any(word in nome_lower for word in termos_acessorio):
                   categoria_id = mapa_categorias.get('acessório')
               elif 'jogo' in nome_lower:
                   categoria_id = mapa_categorias.get('jogo')
               else:
                   categoria_id = mapa_categorias.get('console')
               
               v_venda = limpar_valor(row['venda'])
               v_pix = limpar_valor(row['pix'])
               v_cred = limpar_valor(row['credito'])

               if v_venda == 0 and v_pix == 0 and v_cred == 0:
                   continue

               sql = """
                   INSERT INTO catalogo_mestre 
                   (nome_produto, categoria_id, plataforma, valor_venda_ref, valor_pix_base, valor_cred_base, ativo) 
                   VALUES (%s, %s, %s, %s, %s, %s, %s)
               """
               cursor.execute(sql, (nome, categoria_id, plataforma, v_venda, v_pix, v_cred, True))
               total_inserido += 1
       
       conn.commit()
       print(f"Sucesso! {total_inserido} registros inseridos no catálogo de desenvolvimento.")

   except Error as e:
       print(f"Erro de Banco de Dados: {e}")
   except Exception as e:
       print(f"Erro inesperado: {e}")
   finally:
       if conn and conn.is_connected():
           cursor.close()
           conn.close()
           print("Conexão encerrada com segurança.")

if __name__ == "__main__":
   popular_banco()
