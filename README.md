# MyGames - Plataforma de Compra e Venda de Jogos

## Descrição
Sistema de e-commerce dedicado à negociação de jogos usados, permitindo que clientes realizem a venda de seus produtos diretamente pela plataforma.

## Pré-requisitos
- Python 3.x
- Banco de dados MySQL
- Arquivo `.env` com configurações de banco de dados e chaves de API (fornecido separadamente).
